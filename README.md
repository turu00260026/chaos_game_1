# RAML After Hours

「生活が本体の大人たちが、副業のように世界を救う」を軸にした、
ジャンル汚染・生活負荷・主役度を管理するRPG風アドベンチャーゲームです。

## 遊び方

- GitHub Pages: リポジトリの Pages を有効化すると、ブラウザからそのまま遊べます。
- ローカル: `index.html` をブラウザで開いてください。
- PC / モバイルの両方に対応しています。

## 収録内容

- 全14日・全4章
- 44イベント / 164選択肢
- 8種類のエンディング
- 日ごとの自動セーブと続きから再開
- 周回をまたぐコレクションログ
- キャラクター表情差分、スチール、アルパカ演出

## 操作

- 選択肢: クリック / タップ
- メッセージ送り: クリック / タップ / `Enter` / `Space`

制作室とは分離した、ゲーム単体の公開版です。
